> 这是基于Flutter开发的碎片化阅读App，dd级作品，欢迎初学者学习

hxdm，过来康康啊!

**主要使用的库**

- cached_network_image，图片缓存加载库
- provider，对Widget进行状态管理
- flutter_easyrefresh，非常美观的上拉刷新下拉加载更多动画库
- photo_view，支持图片手势操作的库
- flutter_swiper
- dio
- flutter_spinkit
- 具体看pubspec.yaml...
